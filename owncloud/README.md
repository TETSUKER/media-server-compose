# ownCloud

## ENV

```sh
OWNCLOUD_VERSION=10.16
OWNCLOUD_TRUSTED_DOMAINS=localhost
OWNCLOUD_ADMIN_USERNAME=admin # only for first start
OWNCLOUD_ADMIN_PASSWORD=admin # only for first start
OWNCLOUD_DATA_LOCATION=~/owncloud/files
OWNCLOUD_MYSQL_VERSION=10.11 # minimum required ownCloud version is 10.9
OWNCLOUD_MYSQL_DATA_LOCATION=~/owncloud/db
OWNCLOUD_REDIS_VERSION=6
OWNCLOUD_REDIS_DATA_LOCATION=~/owncloud/redis
OWNCLOUD_DOMAIN=https://owncloud.yourdomain.ru
```
