# Vaultwarden

```sh
VAULTWARDEN_VERSION=latest
VAULTWARDEN_PORT=9445 # Listening port
VAULTWARDEN_SIGNUPS_ALLOWED=true # Controls if new users can register
VAULTWARDEN_DATA_LOCATION=./vaultwarden
VAULTWARDEN_DOMAIN=https://vaultwarden.org # The domain must match the address from where you access the server
```
